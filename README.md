## 🚀 Instalando <Books_Vue_NodeJs>

Para instalar o <Books_Vue_NodeJs>, siga estas etapas:

Instalar o Frontend:
```
cd frontend
npm i
```

Instalar o Backend:
```
cd backend
npm i
```
